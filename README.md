# Разработка интернет-приложений (РИП) 2025

## Расчет энергопотребления различных устройств в квартире

Объектом разработки является система для расчёта энергопотребления различных устройств, позволяющая инженерам-проектировщикам оформлять заявки на расчёт энергопотребления, а консультантам – принимать решение об одобрении или отклонении данных заявок.
